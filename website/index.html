Share opening times with your friends, have a good night's sleep, and still compete.

View on GitHub

How it works

Advent of Code typically rewards those who solve puzzles fastest after they are published (05:00 UTC). aoc-norush levels the playing field by calculating your score based on your net time (from when you open the puzzle) rather than gross time.

How to use it?

  1. Install the browser extension (Chrome, Firefox, etc.).
  2. Open a private leaderboard.
  3. Click on [Ordering] and select the new [NoRush Score] option.
  4. Hover over the table headers for detailed stats.

Browser Support

  • Firefox & Chrome: Supported via store or manual install.
  • Opera & Brave: Install from Chrome Web Store.
